## üîë Key Concepts

### Prayer Types
- **Personal Prayers:** Private, invitation-only
- **Group Prayers:** Visible to group members only
- **Public Prayers:** Open to all users
- **Hidden Prayers:** Impressions concealed until time frame ends
- **Visible Prayers:** Real-time impression sharing

### User Actions
- **Creating Prayers:** Set title, description, time frame, visibility
- **Adding Impressions:** Share divine insights during open prayer period
- **Adding Comments:** Discuss after prayer period closes
- **Adding Feedback:** Prayer creators share outcomes or responses

### Technical Stack
- **Frontend:** Vue.js, SCSS
- **Backend:** Firebase Cloud Functions, Node.js
- **Database:** Firestore
- **Authentication:** Firebase Auth
