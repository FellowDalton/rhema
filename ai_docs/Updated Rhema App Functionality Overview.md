# Updated Rhema App Functionality Overview

## App Purpose
The Rhema App is a web application designed to provide a safe, focused space for users to explore hearing the voice of God together. It aims to replace the use of Facebook groups and Messenger for this purpose, eliminating distractions and limitations of those platforms.

## Core Functionalities

1. Prayer Management
   - Users can create, view, edit, and delete prayers
   - Prayers can be categorized as open or closed
   - Users can set time frames for active prayer periods
   - For hidden prayers, a total impression count is displayed

2. Impression, Comment, and Feedback System
   - During the open prayer period:
     - Users can add "impressions" to prayers, sharing what they hear from God
     - Impressions can be visible in real-time or hidden until the prayer period ends
     - For hidden prayers, a total impression count is shown
   - After the prayer period closes:
     - Users can add comments to discuss the prayer and impressions
   - Feedback:
     - Prayer creators can add feedback as a separate type of response
     - Feedback can include responses from the person or organization being prayed for
   - Impressions, comments, and feedback are all associated with the prayer

3. User Profiles and Dashboards
   - User profiles function as personalized dashboards
   - Dashboards display:
     - All prayers the user is currently part of
     - Prayer history (past prayers)
     - Groups the user has been invited to or has created
   - Users can manage their personal information

4. Group Functionality
   - Users can create prayer groups
   - Only group creators can add members to their groups
   - Users can leave groups they are part of
   - Groups can have private prayers visible only to members

5. Notification System
   - Users receive updates about prayer activities, new impressions, comments, and feedback

6. Search and Filter
   - Users can search for prayers based on various criteria
   - Prayers can be filtered by type, status, or date

## User Types and Capabilities

1. Regular Users
   - Can create personal and public prayers
   - Can be added to prayer groups by group creators
   - Can leave prayer groups they are part of
   - Can add impressions to open prayers they have access to
   - Can add comments to closed prayers they have access to
   - Can view and manage their dashboard, including prayer activities and group memberships
   - Receive notifications about prayers they're involved in

2. Group Administrators (Group Creators)
   - All capabilities of regular users
   - Can create and manage prayer groups
   - Can add users to their groups
   - Can create group-specific prayers

3. App Administrators
   - All capabilities of group administrators
   - Can manage all users and groups
   - Can moderate content across the entire app
   - Have access to app usage statistics and reports

4. Prayer Creators
   - All capabilities of regular users
   - Can add feedback to their created prayers
   - Can manage the visibility and settings of their prayers

## Prayer Types and Visibility

1. Personal Prayers
   - Visible only to the creator and invited participants
   - Can be shared with specific users or groups

2. Group Prayers
   - Visible to all members of a specific group
   - Group admins can manage these prayers

3. Public Prayers
   - Visible to all app users
   - Anyone can participate and comment

4. Hidden Prayers
   - Impressions are not visible during the active prayer period
   - A total impression count is displayed
   - All impressions are revealed when the time frame ends

5. Visible Prayers
   - Impressions are visible in real-time as they are added

## Key Features and User Actions

1. Creating Prayers
   - Users can set a title, description, and time frame
   - They can choose the prayer type (hidden or visible)
   - Users can invite specific participants or groups
   - For hidden prayers, creators can enable the total impression count feature

2. Participating in Prayers
   - During the open prayer period:
     - Users can add impressions sharing what they hear from God
     - They can view others' impressions (depending on prayer visibility)
     - For hidden prayers, users can see the total number of impressions
   - After the prayer period closes:
     - Users can add comments to discuss the prayer and impressions
   - Users can engage in discussions through the comment section

3. Adding Feedback
   - Prayer creators can add feedback as a separate type of response
   - Feedback can include updates or responses from the prayer subject
   - Participants can view feedback provided by the prayer creator

4. Time Frame Management
   - Prayer creators can set and manage time frames for active prayer periods
   - Impressions can only be added during the active time frame
   - Comments can be added after the time frame ends

5. Group Interactions
   - Users can create groups for specific prayer focuses
   - Group creators can add members to their groups
   - Users can leave groups they are part of
   - Group members can participate in group-specific prayers

6. Notifications and Updates
   - Users receive alerts about new impressions, comments, and feedback on their prayers
   - Notifications for group activities and prayer updates

7. Search and Discovery
   - Users can find public prayers on topics they're interested in
   - They can search for specific prayers or browse by categories

8. Dashboard Management
   - Users can view all current and past prayers they've participated in
   - They can see their current group memberships
   - Users can track their prayer activity and participation history
   - Quick access to ongoing prayers and option to leave groups

9. Administrative Functions
   - Admins can monitor app usage and user activities
   - They can moderate content (including impressions, comments, and feedback) and manage user accounts

## Conclusion

The Rhema App provides a focused, distraction-free environment for users to engage in collective spiritual practices, specifically centered on listening for divine guidance and sharing those experiences within a trusted community. It offers a range of privacy options and controlled group functionalities to cater to various user needs and preferences in their spiritual journey. 

The personalized dashboard feature enhances user engagement by providing a comprehensive overview of their prayer activities and group interactions. The addition of impression counts for hidden prayers allows for anonymity while still providing a sense of community participation. The feedback feature enables prayer creators to close the loop on prayer requests, sharing outcomes or responses from the subjects of prayer.

This structured approach to spiritual communication and community building sets the Rhema App apart as a dedicated platform for those seeking to deepen their prayer life and engage in meaningful spiritual discussions.
